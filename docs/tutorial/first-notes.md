# Your first notes workflow

This page shows a small but realistic noteui workflow.

## 1. Open noteui

Start the app:

```bash
noteui
```

## 2. Create your first note

Press `n` to create a note.

Then press `enter` or `o` to open it in your configured editor.

## 3. Give it a useful title

Write a heading such as:

```md
# Inbox
```

noteui uses headings and note content to improve browsing and naming behavior.

## 4. Search it

Back in noteui, press `/` and search for a word from the note.

Search checks:

- note title
- note path
- preview text
- tags

## 5. Organize it

Useful next actions:

- `C`: create a category
- `m`: move the note
- `R`: rename it
- `p`: pin it

## 6. Try a temporary note

Press `N` to create a temporary note.

Temporary notes are useful for quick capture and drafts that you do not want mixed into your main notes immediately.

## 7. Try a todo note

Press `T` to create a todo note template.

This is a good way to see how noteui handles structured checkbox-style notes directly in the terminal workflow.

!!! tip

    Press `?` any time inside noteui to open the full help view.
