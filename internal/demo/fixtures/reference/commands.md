# Command reference

## Navigation

| Key | Action |
|-----|--------|
| `j` / `k` | Move down / up |
| `gg` / `G` | Jump to top / bottom |
| `enter` | Open note in editor |
| `tab` | Switch focus to preview |

## Notes

| Key | Action |
|-----|--------|
| `n` | New note |
| `r` | Rename note |
| `d` | Delete note |
| `m` | Move note |
| `H` | View history |

## Modes

| Key | Action |
|-----|--------|
| `T` | Toggle temp notes view |
| `/` | Search |
| `?` | Help |
| `q` | Quit |
